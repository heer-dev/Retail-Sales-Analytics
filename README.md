# Retail Sales & Customer Buying Behavior Analysis

## Overview

This project analyzes retail sales data to understand customer buying behavior, sales trends, and market share using Python and Power BI.

---

## Workflow

* Data Cleaning & Processing (Python)
* Exploratory Data Analysis (EDA)
* Outlier Handling
* Data Modeling
* Dashboard Development (Power BI)

## Key Insights

* Identified top and bottom performing categories
* Analyzed sales trends across time (CY vs PY vs 3Y)
* Evaluated weekly sales and growth patterns
* Measured market share changes
